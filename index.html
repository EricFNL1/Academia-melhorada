Minhas Perguntas Frequentes

Por que meus boletos não estão sendo registrados?
Se os boletos não estão sendo registrados, siga estas etapas para identificar o problema:
  • Verifique se os dados do usuário da unidade associada ao boleto estão corretos.
  • Confirme se o CPF do usuário está válido e cadastrado corretamente.
  • Garanta que o endereço da unidade está completo e atualizado no sistema.
  • Caso o problema persista, entre em contato com o suporte técnico para análise mais detalhada.
Por que alguns condomínios não aparecem para mim?
Se você não está visualizando todos os condomínios esperados, siga estas orientações:
  • Acesse o módulo Segurança e vá para a opção Incluir Usuários.
  • No perfil do seu usuário, verifique se há algum condomínio selecionado na aba de condomínios.
  • Se houver seleções, remova todas, salve as alterações e volte à tela inicial para verificar se os condomínios estão visíveis.
  • Certifique-se de que as permissões de acesso aos condomínios estão configuradas corretamente.
Por que estou enfrentando problemas visuais no sistema?
Problemas visuais no sistema podem ocorrer devido a atualizações do navegador ou ao armazenamento de informações desatualizadas no cache. Siga estas etapas para solucionar:
  • Pressione as teclas CTRL + F5 para limpar o cache da página e recarregar os dados mais recentes do sistema.
  • Verifique se o navegador que você está utilizando é compatível com o sistema.
  • Certifique-se de que seu navegador está atualizado para a versão mais recente.
  • Se o problema persistir, entre em contato com o suporte técnico para assistência adicional.

Como resetar minha senha no sistema?
Se você precisa resetar sua senha, siga estas instruções:
  • Acesse a página de login do sistema e clique em "Esqueci minha senha".
  • Digite seu e-mail cadastrado e clique em "Enviar".
  • Verifique seu e-mail e siga o link para redefinir sua senha.
  • Caso não receba o e-mail, verifique sua caixa de spam ou entre em contato com o suporte técnico.
O sistema não está gerando relatórios corretamente, o que fazer?
Caso o sistema não esteja gerando relatórios corretamente, siga estas etapas:
  • Verifique se os filtros aplicados estão corretos.
  • Tente gerar um relatório com menos dados para verificar se o problema persiste.
  • Se houver erro, tente atualizar a página e gerar novamente.
  • Entre em contato com o suporte técnico caso o problema continue.
Os e-mails de notificação não estão chegando, como corrigir?
Caso você não esteja recebendo e-mails do sistema, siga estes passos:
  • Verifique se seu e-mail está cadastrado corretamente no sistema.
  • Cheque sua caixa de spam para garantir que os e-mails não estão sendo filtrados.
  • Se o problema persistir, entre em contato com o suporte técnico para verificar configurações do servidor de e-mails.
