Contas a Pagar

Melhore a gestão das suas despesas

Neste treinamento, você aprenderá os principais conceitos sobre o controle de contas a pagar: como registrar pagamentos, acompanhar despesas e gerar relatórios financeiros no módulo do sistema Point Condomínio.

Menu de Acesso

Para começar, acesse o menu Módulo Financeiro → Contas a Pagar.

Selecionando o Condomínio

Ao entrar no menu de contas a pagar, lembre-se de verificar se o condomínio selecionado no campo de seleção é o condomínio em que você deseja acessar esse módulo...

Filtros

Utilize os filtros para exibir as despesas de acordo com sua necessidade. Situação, Tipo de Data, Período, Conta bancária, Plano de contas...

NOTA: Para expandir os filtros, clique no botão "Filtrar"...

Criar uma despesa

Para incluir uma nova despesa, clique em "Adicionar Despesa"...

  • Fornecedor: Cadastre ou informe a quem este valor será pago;
  • Descrição: Informe uma descrição que identifique esta despesa;
  • ...
Liquidação da despesa

Para baixar um título, clique no botão "Pagar" na linha correspondente...

Estorno da baixa

Caso deseje estornar uma baixa realizada, clique no botão de "Engrenagem"...

Remessa e retorno de pagamento

A remessa e retorno de pagamento facilita a vida dos administradores...

Quando a despesa for cadastrada, preencha corretamente o campo "Número do Documento / Cheque / Linha Digitável"...
